Importera en GeneWeb källfil

Skriv in:
  • Det kompletta namnet för din GeneWeb källfil (den måste sluta med ".gw"); du kan använda bläddra knappen för hitta den på din dator, om din webbläsare visar någon.

  • Namnet du vill kalla din GeneWeb databas. Om du inte skriver någonting, kommer den att bli härlätt från GeneWeb källfilen.

Välj de optioner du vill ha:

    Visa en statistik på slutet (de kommer att synas i utskriften).
    Kontrollera inte att databasens data är ok.
    Avluta med att beräkna blodsbanden.

Klicka sedan på "Ok".
