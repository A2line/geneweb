Bienvenido a la página de configuración de GeneWeb

GeneWeb va ha permitirle de explorar las bases de datos genealógicos utilizando su explorador de Web. Si Ud. está conectado al internet, el programa permitirá a todo el mundo de consultar sus datos genealógicos.

Para los debutantes, he aquí un método simple para comenzar.
¿Quizás Ud desee recuperar una base que proviene de una versión anterior de GeneWeb?

Si no, consulte Ud. el menú principal.

Ud puede igualmente consultar la documentación completa de GeneWeb.
