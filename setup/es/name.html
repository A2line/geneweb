Nombre

el nombre de una base de datos esta constituido de letras (mayúsculas o minúsculas), de cifras y del carácter "-". Cualquier otro carácter (particularmente el espacio) es prohibido.

Si Ud. no sabe que nombre escoger, utilize su apellido, o, si este es muy largo, sus primeras letras; o un seudónimo que Ud: utilize habitualmente. Cualquier nombre es valido siempre y cuando la regla mencionada anteriormente sea respetada.
