Nombre incorrecto

Ud. no puede escoger "$o" como un nombre para su base de datos debido a que él contiene uno o varios carácteres prohibidos.

El nombre de una base de datos debe estar constituida de letras, (mayúsculas o minúsculas) sin ningún tipo de acento, de cifras y del carácter "-". Todos los otros carácteres (en particular el "espacio") son prohibidos.
