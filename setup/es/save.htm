Copia de Seguridad y Restauración

Para efectuar una copia de seguridad de su base de datos, aplique Ud. la extracción de un fichero fuente GeneWeb. Copie luego el fichero asi obtenido en alguna otra parte del disco duro, o mucho mejor en un soporte externo (disquete por ejemplo).

Para restaurar una base de datos, aplique Ud. la creación a partir de un fichero fuente GeneWeb.

Ud. puede regrezar al menú principal.
