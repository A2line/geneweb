Import of a GeneWeb source file

Enter:
  • The complete name of your GeneWeb source file (ending with ".gw"); you can use the browse button to find it in your computer, if your browser display one.

  • The name you want to give to your GeneWeb data base. If you write nothing, it will be deducted from the GeneWeb source file.

Select the options you want:

    Display some statistics at the end (they will be visible in the traces).
    Do not check the consistency of the data.
    End by the initialization of consanguinities.

Then click on "Ok".
