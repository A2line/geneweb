Recover a data base

$v{Warning: there already exists a data base named "$o" in the destination directory. If you push the button below, it will be erased. If you just made this handling, it is probably its result, therefore it is not important. Otherwise, go back and choose another name for your data base.

} Starting directory and data base: 

  $a
  $i
Destination directory and data base: 
  $w
  $o

Warning: the operation can take several seconds or minutes, depending on the case. Be patient.

Push the below button to launch the operation$v{, and overwrite the data base "$o" of the destination directory}.

$h

For information, this operation corresponds to the following commands: 

     $$ cd $a
     $$ .$/$U $i $O $w$/$T
     $$ cd $w
     $$ .$/$C $T -o $o > comm.log
