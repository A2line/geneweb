Error

The previous command exited with an error code. Your data base "$o" may have not been created. Look at the traces, maybe you can find an explanation.

You may have come from a too old version of GeneWeb. In this case, try again to do the transfer using the option to "pass by a GEDCOM file" (you can also go back to the previous screen to do it).

If you were already doing that, and it does not work better, ask the creator of the GeneWeb program. Send him the traces of the command.

You can return to the main menu.
