Parameters updated

Here is an access to your data base in order to test your parameters changing:

  • http://localhost:2317/$a

For information:

  • The parameters are recorded in the text file "$a.gwf" in the directory "gw" of your GeneWeb distribution.
  • If you gave a text to put in the bottom of the pages, this text is recorded in the file named "$a.trl" in the subdirectory "lang" of the directory "gw".

You can return to the main menu.
