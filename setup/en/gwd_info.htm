Service gwd

To start the service:

  • Under Windows, launch the program "gwd".

  • Under Unix, launch an xterm and type the following commands: 

         $$ cd $w
     $$ $x$/gwd -hd $x

  • Under Linux, if you installed GeneWeb as an rpm package, the service has been automatically launched. If you stopped it and want to restart it, become root and type: 

         $$ /etc/rc.d/init.d/gwd start

To stop the service:

  • Under Windows or Unix, type "control C" in the window where it is running.

  • Under Linux, if you installed GeneWeb as an rpm package, become root and type: 

         $$ /etc/rc.d/init.d/gwd stop
