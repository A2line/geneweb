Parameters for "gwd"

Here are the parameters of the command "gwd". Change the one you estimate necessary:

-lang Default language.

By default, all the pages of the databases are displayed in this language. Select your preferred language.

For a given database, you can still change this choice by configurating the specific parameters of this database. And in the welcome page of the base, you can still click on another flag to select another language. The present choice is therefore like a "default default" language.

If "-" is selected, it is French.

 
-only Only authorized address.

Only the requests from this Internet address are accepted.

To know which value put for your computer, look at the traces of the command "gwd" which are displayed each time you navigate in your databases. You have to put what you see behind the "From:".

 
-log Connection log file.

By default the connection traces appear in the window of "gwd". Each time you navigate, or somebody accesses to one of your database, 4 or 5 lines are displayed.

If you want that the traces go rather in a file, write here the name of the file you wish. This file will be created if it does not exist, in the main directory of GeneWeb where you see the commands "setup" and "gwd". It will be extented à each request.

 
Other parameters exist for the command "gwd", but are not configurable here because they can perturbate the operation of "setup".

Indeed, the navigation by "setup" is mainly destinated to make the use of GeneWeb simpler, but does not allow to exploit quite all the possibilities. To find the other parameters and their use, you must be a little bit "hacker": some informations can be found in the documentation.
