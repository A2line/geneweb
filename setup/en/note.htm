Note

Some Web browsers, which propose a browse button, do not allow to select directories but only files: indeed, by clicking on a directory, they give you the list of the files inside, but it does not allow you to "exit" with the directory itself.

If it is the case of your Web browser, to select your directory, select any file inside it. The program "gwsetup" understands that it is the directory you are talking about.

Another solution, if the directory is empty or holds only directories, is to select a file close to the directory, to accept the browsing and then afterwards in the input area fix the good access path to the directory.
