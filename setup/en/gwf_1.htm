Parameters "$a"

Here are the parameters of the base $a. Change what you estimate necessary:

- body_prop -
Allows to change the background color, put a background pattern, or change the color of the text or the links. Its syntax is the one of HTML accepted in the tag "<body>" before the ">". See your HTML documentation. Example: 

     text=#CC0000 vlink=yellow

Some interesting variables...

  • background: background image (see the note)
  • bgcolor: background color
  • text: text color
  • link: links color
  • vlink: color of already visited links

You can also select one of the proposed colors combinations.

This applies not only to the welcome page but to all the displayed pages for this data base.

- default_lang -
Default language.

If it is "-", the default language of the command "gwd" is applied.

- friend_passwd -
Password for "friends".

"Friends" can see all dates, notes and photos of the persons born since less than one century. The ones who have not entered the below password will not see these informations.

If this area is empty, everybody is a "friend".

- wizard_passwd -
Password for "wizards".

"Wizards" can moreover make modifications in the data base.

If this area is empty, everybody is a "wizard".

- can_send_image -
Ability to send images.

If yes, the "wizards" are authorized to upload images.

Can be useful in local, in order to be able to put an image for somebody, because the procedure installs automatically the image at the right place with the appropriate file name.

Yes No
- wizard_just_friend -
"Wizards" become simple "friend".

If yes, the "wizards" loose their power to make updates.

This can be useful, temporarilly, when you make a cleanup of your data base, if it is accessible to Internet, to avoid that "wizards" make changes which would be lost during the operation (remember to suppress this option after cleanup).

Yes No
- renamed -
Base has been renamed.

If you changed the name of your data base, but this data base has been existed since a long time and you think that many people made links to it, keep this parameter file for this base and write here the new name of the data base. People who consult the data base with the old name receive then a page indicating them the new name to use with the same clickable request.

If this area is empty, the data base has not been renamed.

If you want that a text appears in the bottom of each page, write it here. You can use HTML tags.
