Paramètres mis-à-jour

Voici un accès à votre base pour tester vos changements de paramètres.

  • http://localhost:2317/$a

Pour information:

  • Les paramètres sont enregistrés dans le fichier texte "$a.gwf" dans le répertoire "gw" de votre distribution GeneWeb.
  • Si vous avez donné un texte à mettre au bas des pages, ce texte est enregistré dans la fichier de nom "$a.trl" dans le sous-répertoire "lang" du répertoire "gw".

Vous pouvez revenir au menu principal.
