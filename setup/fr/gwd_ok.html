Paramètres mis-à-jour

Information: les paramètres sont enregistrés dans le fichier texte "gwd.arg" dans le répertoire "gw" de votre distribution GeneWeb.

Voici son contenu: 

$r{[fichier vide]}
Si le service gwd est lancé, il faut l'arrêter (en tapant "control C" dans la fenêtre où il tourne) et le relancer pour que les nouveaux paramètres soient pris en compte.

Vous pouvez revenir au menu principal.
