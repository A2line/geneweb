Sauvegarde et restauration


Pour sauvegarder votre base de données, appliquez l'extraction d'un fichier source GeneWeb. Puis copiez le fichier ainsi obtenu autre part sur le disque, ou mieux, sur un support externe (disquette par exemple).

Pour restaurer une base de données, appliquez la création à partir d'un fichier source GeneWeb.

Vous pouvez revenir au menu principal.
