Paramètres mis-à-jour

Pour que les paramètres soient pris en compte, il faut arrêter le service gwd, s'il tourne, et le relancer.

Information: les paramètres sont enregistrés dans le fichier texte "gwd.arg" qui se trouve dans le répertoire "gw" de votre distribution GeneWeb.

Voici son contenu: 

$r{[fichier vide]}
Vous pouvez revenir au menu principal.
