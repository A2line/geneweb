Application de "$d"

L'opération "$d" va être appliquée sur votre base "$a". Attention: l'opération peut prendre quelques secondes ou quelques minutes, suivant les cas. Soyez patient.

Appuyez sur le bouton ci-dessous pour démarrer:

$h

Pour information, cette opération correspond à la suite des commandes suivantes: 

     $$ cd $w
     $$ $c$p > comm.log
