Récupérer une base

$v{Attention: il existe déjà une base de nom "$o" dans le répertoire d'arrivée. Si vous appuyez sur le bouton ci-dessous, cela va l'effacer. Si vous venez déjà de faire cette manipulation, il s'agit probablement de son résultat, ce n'est donc pas grave. Sinon, revenez en arrière et choisissez un autre nom pour votre base.

} Répertoire et base de départ: 

  $a
  $i
Répertoire et base d'arrivée: 
  $w
  $o

Attention: l'opération peut prendre quelques secondes ou quelques minutes, suivant les cas. Soyez patient.

Appuyez sur le bouton ci-dessous pour lancer l'opération$v{ et écraser la base "$o" du répertoire d'arrivée}:

$h

Pour information, cette opération correspond à la suite des commandes suivantes: 

     $$ cd $a
     $$ .$/$U $i $O $w$/$T
     $$ cd $w
     $$ .$/$C $T -o $o > comm.log
