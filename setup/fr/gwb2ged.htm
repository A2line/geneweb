Gwb2ged:
Extraction d'un fichier gedcom

Sélectionnez votre base de données:

$b{| }
   $a
- il n'y a aucune base de données actuellement

Entrez le nom que vous voulez donner à votre fichier GEDCOM. Si vous ne mettez rien, ce sera le nom de la base suivi du suffixe ".ged".

Sélectionnez les options que vous désirez:

    Coder les caractères en ASCII.
    Coder les caractères en ANSEL.

    Économiser l'espace mémoire pendant l'opération (mais ce sera plus lent).

 N'extraire que les ancêtres de...
PrénomNum
Nom

 N'extraire que les descendants de...
PrénomNum
Nom

Si vous utilisez les deux options ci-dessus, c'est l'intersection qui comptera: personnes à la fois ancêtres de l'un et descendants de l'autre.
