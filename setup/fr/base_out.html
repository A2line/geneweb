Création d'une base de données

$v{Attention: il existe déjà une base de nom "$o" dans votre répertoire. Si vous appuyez sur le bouton "Ok" ci-dessous, cela va l'effacer.

Si vous venez déjà de faire cette manipulation, il s'agit probablement de son résultat, ce n'est donc pas grave. Sinon, revenez en arrière et choisissez un autre nom pour votre base.

} Nous allons créer une base de nom "$o". Remarque: cette opération peut s'effectuer en un clin d'oeil ou prendre quelques dizaines de secondes ou quelques minutes, suivant les cas. Soyez patient.

Appuyez sur le bouton ci-dessous pour créer votre base de données$v{ et écraser la base "$o" du répertoire d'arrivée}:

$h

Pour information, cette opération correspond à la suite des commandes suivantes: 

     $$ cd $w
     $$ $c$p > comm.log
Cela fabriquera un répertoire de nom "$o.gwb".
