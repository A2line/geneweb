Démarrage

Pour démarrer, vous avez besoin d'une base de données GeneWeb.

  • Si vous partez de rien, vous devez créer une base de données vide. Cliquez ici.

  • Si vous disposez d'un fichier GEDCOM, vous devez le convertir en base de données GeneWeb. Cliquez ici.

Pour revenir à l'écran précédent, utilisez la touche "retour en arrière" de votre navigateur.
