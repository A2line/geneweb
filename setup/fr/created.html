Base de données "$o" créée

Bravo. Votre base de données "$o" est créée.

Pour consulter votre base de données, il faut que le service gwd soit lancé. Assurez-vous qu'il l'est.

Puis essayez sur une des adresses ci-dessous. La page affichée s'appelle "page d'accueil" de votre base de données. Mettez-la dans vos signets.

  • http://localhost:2317/$o
  • http://127.0.0.1:2317/$o

Si aucune de ces adresses ne fonctionne, demandez conseil à d'autres utilisateurs de GeneWeb ou au créateur du programme.

Vous aurez peut-être besoin de consulter le mode d'emploi des modifications. Conseil: prenez une autre fenêtre de votre navigateur pour la consulter.

Pour diverses manipulations de votre base de données ou la creation d'autres bases de données, voyez le menu principal.

Pour revenir à l'écran précédent, utilisez la touche "retour en arrière" de votre navigateur.
