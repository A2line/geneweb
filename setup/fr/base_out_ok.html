Base de données créée

Voilà. L'opération est terminée. Votre base de données de nom "$o" est créée. Si l'opération a écrit des messages (traces), vous pouvez les consulter ici.

Pour la consulter et éventuellement la modifier, il faut que le service gwd soit lancé.

S'il l'est, essayez l'adresse ci-dessous. La page affichée s'appelle "page d'accueil" de votre base de données. Mettez-la dans vos signets.

  • http://localhost:2317/$o

Vous aurez peut-être besoin de consulter le mode d'emploi des modifications.

Ou vous pouvez revenir au menu principal.
