Attention: conflit

Le programme vient de détecter qu'il existe déjà un répertoire de nom "$o.gwb" contenant probablement déjà une base de données GeneWeb de nom "$o".

Vous pouvez décider:

  • de continuer en écrasant ce répertoire, ou
  • de donner un autre nom à votre base de données.

Si vous veniez juste de fabriquer votre base de données GeneWeb et que vous êtes en train de faire une autre tentative, il se peut qu'il s'agisse de votre tentative précédente: dans ce cas, vous pouvez écraser le répertoire sans vous inquiéter.

Pour continuer en écrasant le répertoire "$o.gwb", cliquez ici.

Votre base "$o" va être effectivement créée par la commande "$d". Attention: l'opération peut répondre immédiatement ou prendre plusieurs dizaines de secondes, suivant les cas. Soyez patient.

Sinon revenez à l'écran précédent avec la touche "retour en arrière" de votre navigateur et choisissez un autre nom.

Pour information, l'opération que vous allez faire en cliquant ci-dessus correspond à la suite de commandes intéractives:

> cd $w
> $c$p

Cela fabriquera un répertoire de nom "$o.gwb" en écrasant l'ancien.
