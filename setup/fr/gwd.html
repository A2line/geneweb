Paramètres "gwd"

Configurez les paramètres que vous désirez:

-hd Répertoire de GeneWeb

C'est là où "gwd" doit trouver les répertoires "lang" et "images" nécéssaires à son fonctionnement normal. Ce répertoire est soit absolu, soit relatif à l'endroit où "gwd" est installé.

Dans la présente distribution, "gwd" est installé dans le répertoire "gw". La version "gwd" qui existe au premier niveau se contentant d'appeler ce "gwd"-là.

-bd Répertoire des bases de données.

C'est là où "gwd" doit trouver les répertoires contenant les bases de données. Ce répertoire est soit absolu, soit relatif à l'endroit où "gwd" est installé.

Dans la présente distribution, "gwd" est installé dans le répertoire "gw". La version "gwd" qui existe au premier niveau se contentant d'appeler ce "gwd"-là.
