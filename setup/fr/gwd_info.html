Service "gwd"

Pour lancer ce service:

  • Sous Windows, double-cliquez sur l'icône de de "gwd".

  • Vous pouvez aussi lancer une fenêtre de commandes intéractives (MSdos sous Windows ou Xterm sous Unix) et taper les commandes suivantes: 

         $$ cd $u
     $$ .$/gwd

Cela doit afficher quelques lignes qui se terminent par:

     Type control C to stop the service
     Ready JJ/MM/AAAA HH:MM port 2317...

Ne lancez ce service qu'une seule fois. Si vous désirez l'arrêter, tapez "control-c" dans la fenêtre où il s'exécute (comme indiqué).
