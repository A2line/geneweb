Service gwd

Pour démarrer le service:

  • Sous Windows, double-cliquez sur l'icône de "gwd", puis revenez ici.

  • Sous Unix, lancez un xterm et tapez les commandes suivantes: 

         $$ cd $u
     $$ .$/gwd
  • Sous Linux, si vous avez installé GeneWeb à partir un paquetage rpm, le service a été automatiquement lancé. Si vous l'avez arrêté et voulez le relancer, devenez root et tapez: 

         $$ /etc/rc.d/init.d/gwd start

Pour arrêter le service:

  • Sous Windows ou Unix, tapez "control C" dans la fenêtre où il tourne

  • Sous Linux, si vous avez installé GeneWeb par un paquetage rpm, devenez root et tapez: 

         $$ /etc/rc.d/init.d/gwd stop
