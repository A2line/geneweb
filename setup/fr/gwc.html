Importation d'un fichier source GeneWeb

Entrez:
  • Le nom complet de votre fichier source GeneWeb (se terminant par ".gw"); vous pouvez utiliser le bouton de navigation pour le rechercher sur votre ordinateur, si votre navigateur en affiche un.

  • Le nom que vous voulez donner à votre base de données GeneWeb. Si vous ne mettez rien, il sera déduit du nom du fichier source GeneWeb.

Sélectionnez les options que vous désirez:

    Afficher quelques statistiques à la fin (elles seront visibles dans les traces).
    Ne pas vérifier la cohérence des données.
    Terminer par le précalcul de consanguinité.

Puis cliquez sur "Ok".
