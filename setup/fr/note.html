Note

Les navigateurs Web proposent souvent un bouton de navigation pour sélectionner un fichier. Mais certains ne permettent pas de sélectionner un répertoire.

Si c'est le cas du vôtre, pour sélectionner votre répertoire, sélectionnez n'importe quel fichier qui s'y trouve. Le programme "setup" comprendra que c'est bien du répertoire que vous parlez.
