Importation d'un fichier GEDCOM

Entrez:
  • Le nom complet de votre fichier GEDCOM; vous pouvez utiliser le bouton de navigation pour le rechercher sur votre ordinateur.

  • Le nom que vous voulez donner à votre base de données GeneWeb.

Puis cliquez sur "Ok".

Pour revenir à l'écran précédent, utilisez la touche "retour en arrière" de votre navigateur.
