Renommage d'une base de données

Vos bases de données se trouvent dans le répertoire "gw" de votre distribution.

Une base de données GeneWeb est un répertoire avec le suffixe ".gwb". Pour renommer une base, renommez simplement ce répertoire.

Exemple: si vous voulez renommer votre base "toto" en "titi", renommez le répertoire "toto.gwb" en "titi.gwb".

Vous pouvez revenir au menu principal.
