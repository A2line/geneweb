Nettoyage d'une base de données

Si vous avez fait beaucoup de modifications dans votre base de données, ou si vous avez simplement entré beaucoup de données, il est conseillé de faire un nettoyage, surtout si vous constatez que l'accès à votre base de données commence à ralentir.

Pour un nettoyage simple, appliquez le programme d'initialisation des consanguinités.

Pour un nettoyage en profondeur, appliquez les opérations suivantes:

  • Renommez votre base de données. Cela servira en plus de sauvegarde.
  • Extrayez un fichier source GeneWeb de votre base de données.
  • Reconstruisez votre base de données à partir de ce fichier source.

Ou vous pouvez revenir au menu principal.
