Création d'une base de données

D'accord. Nous allons créer une base de nom "$o". Remarque: cette opération peut s'effectuer en un clin d'oeil ou prendre quelques dizaines de secondes, suivant les cas. Soyez patient.

Appuyez sur le bouton ci-dessous pour créer votre base de données:

$h

Pour information, cette opération correspond à la suite des commandes suivantes:

$$ cd $w
$$ $c$p > comm.log
Cela fabriquera un répertoire de nom "$o.gwb".
