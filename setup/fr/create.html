Création de la base "$o"

Bien. Le nom que vous avez donné est correct. Le programme a détecté qu'il n'existait aucune base de données GeneWeb portant déjà ce nom et vous ne risquez donc pas de perdre des données.

En cliquant ici, votre base "$o" va être effectivement créée par la commande "$d".

Attention: l'opération peut répondre immédiatement ou prendre quelques dizaines de secondes. Soyez patient.

Pour revenir à l'écran précédent, utilisez la touche "retour en arrière" de votre navigateur.

Pour information, la création effective de votre base correspond à la suite de commandes intéractives:

> cd $w
> $c$p

Cela fabriquera un répertoire de nom "$o.gwb" qui est votre base de données GeneWeb.
