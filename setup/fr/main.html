Menu principal

Vous pouvez:

  • Accéder à une base de données GeneWeb existante.
  • Créer une base de données vide.
  • Créer une base de données à partir d'un fichier GEDCOM (paramètres par défaut).
  • Créer une base de données à partir d'un fichier GEDCOM (parametrables).
  • Créer une base de données à partir de fichiers sources GeneWeb.
  • Passer le programme d'initialisation des consanguinités dans une base de données.
  • Extraire un fichier GEDCOM d'une base de données.
  • Extraire des fichiers sources GeneWeb d'une base de données.
  • Changer le nom d'une base de données.
  • Configurer les paramètres d'une base de données.
  • Configurer le service "gwd".

Pour revenir à l'écran précédent, utilisez la touche "retour en arrière" de votre navigateur.
