Suppression de bases de données

Votre programme "setup" ne veut pas prendre l'initiative de détruire lui-même une base de données.

Si vous voulez le faire, regardez dans le répertoire "gw" de votre distribution. Une base de données GeneWeb est un répertoire avec le suffixe ".gwb".

Exemple: la base de données "toto" est un répertoire portant le nom "toto.gwb".

Pour supprimer votre base, mettez ce répertoire à la poubelle ou supprimez-le par le moyen habituel de votre système d'exploitation.

Vous pouvez revenir au menu principal.
