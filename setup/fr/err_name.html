Nom incorrect

Vous ne pouvez pas choisir "$o" comme nom de bases de données car il contient des caractères interdits.

Le nom d'une base de données est constitué de lettres (majuscules ou minuscules), de chiffres et du caractère "-". Tout autre caractère (en particulier l'espace) est interdit.

Si c'est votre première base de données GeneWeb et que vous ne savez pas quoi choisir comme nom, prenez votre nom de famille, ou, s'il est trop long, ses premières lettres, ou un pseudonyme que vous avez l'habitude d'utiliser. N'importe quel nom est valable pourvu qu'il respecte la règle ci-dessus. Prenez un nom pas trop long de préférence.
