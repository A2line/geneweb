Image de fond

Pour indiquer une image de fond pour la base "$o", il faut la mettre sous la forme (en remplaçant "toto.jpg" par le vrai nom de votre fichier image: 
     background="gwd$t{.exe}?m=IM;b=$o;v=/toto.jpg"

Mettez ce fichier image dans le répertoire (que vous devrez éventuellement créer): 

     gw$/images$/$o
